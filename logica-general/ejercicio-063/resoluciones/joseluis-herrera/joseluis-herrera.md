# Explicación
Este programa funciona como un sistema de gestión de clasificaciones para un torneo de esports, evaluando automáticamente la elegibilidad de cada jugador. Al contrastar el desempeño competitivo en puntos y rachas con las condiciones técnicas de latencia, el sistema organiza el flujo del torneo de manera justa, permitiendo filtrar a los participantes según criterios de rendimiento y calidad de conexión.
