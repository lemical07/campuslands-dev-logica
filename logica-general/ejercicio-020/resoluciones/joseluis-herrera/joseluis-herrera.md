# Ejercicio 20

# Explicación
En este ejercicio se implementa un algoritmo de filtrado inteligente para un puesto de comida urbana. El objetivo es estructurar una estrategia de selección que evalúe las opciones de un menú dinámico, descartando de forma automática los platillos que superen el presupuesto del cliente o que no cumplan con sus restricciones de dieta.
