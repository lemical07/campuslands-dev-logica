# Ejercicio 10

## Explicacion
En el siguiente ejercicio se realiza un sistema para poder sacar un passapaporte en donde se verifica si el usuario cumple con los requisitos que se solicitan o si no los cumple como los requisistos para sacar el pasaporte o si el peso es el adecuado si no tiene que pagar una multa, si es que llega a cumplir todos los requisitos se le da su pasaporte y se le desea un felix viaje.
