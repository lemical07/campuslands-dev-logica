# Ejercicio 21

# Explicación
En este ejercicio se diseña un sistema de clasificación de rangos para una liga de videojuegos competitivos (Esports). El objetivo es procesar el desempeño de un jugador combinando sus estadísticas de combate y victorias para calcular un puntaje de estrategia, aplicando reglas estrictas y filtros de conducta para determinar su división final.
