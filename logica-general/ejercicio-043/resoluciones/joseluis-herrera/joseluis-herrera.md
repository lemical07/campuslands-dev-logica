# Ejercicio 43

# Explicacion 
En este ejercicio se implementa un motor de decisiones para el arbitraje automático de un torneo de esports. El objetivo es estructurar una lógica de resolución de disputas que evalúe las incidencias reportadas en una partida (como desconexiones, uso de glitches o conducta antideportiva). El sistema debe decidir automáticamente entre: validar el resultado actual, forzar un "rematch" (partida de repetición) o descalificar al equipo infractor, basándose en la gravedad de los hechos y el momento en que ocurrieron.
