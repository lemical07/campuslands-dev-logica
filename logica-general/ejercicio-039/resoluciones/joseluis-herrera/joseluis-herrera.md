# Ejercicio 39 

# Explicacion 
En este ejercicio se implementa un algoritmo de gestión de inventario para un laboratorio de química. El objetivo es estructurar una estrategia de control de stock que evalúe los niveles de reactivos almacenados, aplicando reglas de seguridad (caducidad y peligrosidad) para determinar si un compuesto debe ser reabastecido de forma prioritaria, garantizando así la continuidad operativa y el cumplimiento de las normas de almacenamiento.
