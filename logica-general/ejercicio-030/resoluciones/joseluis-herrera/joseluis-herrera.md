# Ejercicio 30

# Explicación
En este ejercicio se automatiza el proceso de lectura y aplicación de políticas de una agencia de viajes y turismo. El objetivo es estructurar una función que interprete de forma secuencial una serie de instrucciones y restricciones contractuales (edad del pasajero, destino, duración y tipo de temporada) para validar la viabilidad del viaje y calcular la tarifa final.
