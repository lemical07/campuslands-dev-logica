# Ejercicio 05

## Explicacion
En el siguiente ejercicio se realiza una se hace una verificacion para saber cuales que clientes son de mayor, media y baja prioridad para eso se hace una funcion en donde se hace un recorriod y podamos observar que tipo de prioridad tiene cada cliente.  La regla que se usa aqui es que almenos tiene aver un cliente si no de lo contrario se dira que no hay clientes. 

## Caso normal 
```text
  {  
    cliente: "Maria",
    vehiculo: "Bus",
    prioridad: "Alta"
  }
```

Resultado esperado:

Clientes de mayor prioridad:  Maria

### Caso borde
```text
    {
    }
```

Resultado esperado:
No hay cliente en el taller
