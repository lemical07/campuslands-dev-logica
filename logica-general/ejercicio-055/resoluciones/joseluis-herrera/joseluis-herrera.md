# Explicación
Este programa funciona como un validador de flujo de trabajo para artistas digitales, asegurando que cada archivo cumpla con los estándares técnicos antes de finalizar la obra. Mediante la evaluación de capas, tiempos de ejecución y resolución, el sistema detecta inconsistencias que podrían afectar la calidad final o el rendimiento del software, permitiendo correcciones preventivas.
