# Ejercicio 41

# Explicación
En este ejercicio se implementa un algoritmo de validación de datos para una liga de fútbol sala. El objetivo es estructurar una estrategia de verificación que procese el rendimiento de los equipos, asegurando que los registros (goles, partidos jugados y tarjetas) sean coherentes antes de calcular la posición en la tabla de clasificación. El sistema debe descartar registros mal formados o imposibles (como tarjetas recibidas superiores a los minutos jugados) para mantener la integridad de la liga.
