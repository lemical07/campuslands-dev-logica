# Explicación
En este ejercicio se desarrolla un sistema de auditoría y asignación de rangos para la tabla de posiciones de una liga de fútbol sala. El objetivo es construir filtros de validación secuenciales que limpien los datos ingresados antes de procesar las estadísticas de rendimiento de los equipos, evitando que registros dañados o imposibles alteren el ranking oficial.
