# Explicacion 
Este proyecto consiste en un script en JavaScript diseñado para automatizar la auditoría técnica de archivos y proyectos de ilustración digital dentro de un estudio de diseño, evaluando los formatos según sus DPI (puntos por pulgada) de resolución.


## Estructura de Datos de los Objetos

Cada ilustración en el flujo de trabajo mantiene estrictamente la siguiente sintaxis de ejemplo:

```javascript
{
  nombre: "Diseño_Personaje",
  equipo: "Ilustración",
  puntos: 350,
}
