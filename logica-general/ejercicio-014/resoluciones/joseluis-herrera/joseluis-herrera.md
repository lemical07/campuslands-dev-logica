# Explicacion

Este proyecto consiste en un script de automatización en JavaScript diseñado para auditar, analizar y diagnosticar anomalías en los registros de clientes dentro de un sistema de gestión de un estudio de tatuajes.


## Estructura de Datos de los Objetos

Cada registro dentro del sistema mantiene estrictamente la siguiente sintaxis de ejemplo:

{
  nombre: "Mateo",
  equipo: "España",
  puntos: 88,
}
