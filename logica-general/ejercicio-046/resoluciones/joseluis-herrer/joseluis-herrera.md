# Ejercicio 46

# Explicación
En este ejercicio, desarrollaremos un sistema para localizar el vehículo ideal dentro de un catálogo de autos hiperdeportivos. El reto consiste en pre-procesar (organizar) la información antes de aplicar filtros complejos. El objetivo es identificar qué vehículo maximiza la relación entre potencia y peso, aplicando restricciones de seguridad y exclusividad.
