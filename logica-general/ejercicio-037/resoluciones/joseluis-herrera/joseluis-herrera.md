# Ejercicio 37 

# Resumen 
En este ejercicio se implementa un algoritmo de análisis comparativo para la selección de estructuras en un proyecto de arquitectura 3D. El objetivo es estructurar una estrategia de evaluación que procese un conjunto de diseños dinámicos, calculando de forma automática el índice de eficiencia de cada edificio basado en parámetros técnicos de altura, nivel de detalle y estabilidad estructural, determinando finalmente cuál de ellos es la opción más óptima.
