# Ejercicio 02 

## Explicacion 
Se realizo un ejercicio para una puntuacion de puntos para unos equipos de futbol sala en el cual se verifica que 
equipos cumple con mas de 40 puntos o los 40 puntos para pasar a la semi-final del partido basado en eso se utilizo 
una condicional la cual es un if-else para logica y un forEach donde se utiliza para recorrer la lista de objetos.
