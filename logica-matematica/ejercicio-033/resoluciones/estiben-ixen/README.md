# Resolución: Reto Paracaidismo - JS

## Pensamiento del problema
Utilizamos `Math.sqrt` y `Math.pow` para calcular la hipotenusa entre las coordenadas dadas. La lógica del viento se implementó mediante un operador ternario para mayor legibilidad.

## Reglas aplicadas
- Distancia euclidiana estándar.
- Penalización variable: Si el viento > 15, la distancia percibida aumenta.
- Clasificación en rangos de seguridad.

## Cómo ejecutar
Ejecuta el archivo directamente en tu terminal usando Node.js:
`node nombre-apellido.js`
