# Ejercicio 12
# Explicación
En este ejercicio se analiza el desempeño de un jugador de ping-pong basándose en la efectividad de su servicio. El objetivo es calcular la probabilidad matemática de ganar un punto cada vez que el jugador saca, utilizando una función
que procesa los datos y define si el saque se considera de alto nivel o si necesita mejorar.
