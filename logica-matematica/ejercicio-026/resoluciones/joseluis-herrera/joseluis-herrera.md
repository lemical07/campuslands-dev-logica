# Ejercicio 26
# Explicación
En este ejercicio se desarrolla un algoritmo de búsqueda especializada para un catálogo de autos hiperdeportivos. El objetivo es estructurar una función que examine una base de datos y localice el primer vehículo que rompa una barrera de velocidad específica, use el tipo de motorización deseado y pertenezca a una serie de producción limitada (exclusividad).
