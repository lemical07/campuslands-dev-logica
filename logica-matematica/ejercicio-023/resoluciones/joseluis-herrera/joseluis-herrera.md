# Ejercicio 23

# Explicación
En este ejercicio se procesan las estadísticas de rendimiento de un equipo de Esports a lo largo de un torneo. El objetivo es estructurar una función matemática que extraiga el promedio y la mediana de una lista de puntajes para evaluar la regularidad del equipo, evitando que una sola partida muy mala o muy buena altere por completo el diagnóstico de su nivel competitivo.
