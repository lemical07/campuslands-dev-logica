# Ejercicio 047 - Secuencias numéricas

## ¿Cómo pensé el problema?

Identifiqué que era necesario generar una secuencia ordenada de números que representara una playlist.

## Reglas aplicadas

1. Validar que la cantidad sea mayor que cero.
2. Generar la secuencia utilizando un ciclo.
3. Mostrar la lista generada y la cantidad total de elementos.

## Cómo ejecutar

```bash
node stefani-sanchez.js
```

## Casos probados

### Caso normal

- Inicio: 1
- Cantidad: 10

Resultado:

Se genera una secuencia del 1 al 10.

### Caso borde

- Inicio: 5
- Cantidad: 0

Resultado:

Cantidad inválida.

## Conclusión

Se utilizaron:

- Funciones.
- Arreglos.
- Ciclos.
- Condicionales.
