# Ejercicio 41

# Explicacion
En este ejercicio se implementa un algoritmo de gestión de economía para un videojuego competitivo. El objetivo es estructurar una lógica de control que gestione el presupuesto de los equipos al final de cada ronda. La función debe procesar los ingresos por victoria o derrota, aplicar bonificaciones por rachas y realizar descuentos por penalizaciones de equipo, garantizando que el presupuesto se mantenga dentro de los límites operativos del sistema.
