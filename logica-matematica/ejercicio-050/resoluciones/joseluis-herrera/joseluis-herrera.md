# Ejercicio 50 

# Explicación
En este ejercicio, gestionamos el cálculo de costos para paquetes turísticos internacionales. El desafío principal es manejar la precisión numérica al realizar conversiones de moneda y aplicar impuestos. Al trabajar con precios, el redondeo es crítico: errores mínimos pueden generar discrepancias financieras al escalar el volumen de ventas.
