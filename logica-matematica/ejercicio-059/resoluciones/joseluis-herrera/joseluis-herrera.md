# Explicación
Este programa actúa como un filtro de validación para la captura de fórmulas químicas, asegurando que los datos introducidos sean físicamente coherentes. Al verificar la correspondencia entre los elementos declarados y sus cantidades, el sistema previene errores de entrada que podrían comprometer la precisión de los cálculos estequiométricos posteriores.
