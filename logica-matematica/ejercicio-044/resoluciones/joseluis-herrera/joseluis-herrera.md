# Ejercicio 44

# Explicación
En este ejercicio se implementa un algoritmo de análisis para identificar las unidades más valiosas dentro de un inventario de motocicletas. El objetivo es estructurar una estrategia que recorra el stock, clasifique las unidades según rangos de kilometraje y determine cuál es la moto de mayor valor en cada categoría, asegurando que el inventario se organice de manera jerárquica para la toma de decisiones comerciales.
