# Ejercicio 18

# Explicación
En este ejercicio se evalúa la calidad de una unión de soldadura industrial convirtiendo los puntos de una inspección visual a una escala unificada de 1 a 10. El objetivo es estructurar una función matemática que tome cualquier puntaje bruto y lo ajuste a un formato estándar (normalización) para determinar de forma justa si la soldadura cumple con las normas de seguridad estructural o si debe corregirse.
