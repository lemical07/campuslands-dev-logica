# Explicación
Este programa organiza el rendimiento de diversos proyectos de arquitectura 3D mediante un sistema de ranking basado en puntajes de diseño y sostenibilidad. Al procesar y ordenar automáticamente esta información, el sistema permite visualizar rápidamente los proyectos con mejor desempeño, facilitando la toma de decisiones para futuras inversiones o presentaciones ante clientes.
