# Ejercicio 45 

# Explicación
En este ejercicio se implementa un algoritmo de gestión de inventario para un taller mecánico. El objetivo es estructurar un sistema de control de existencias que identifique qué repuestos han caído por debajo de sus niveles mínimos de seguridad y cuáles han excedido el límite de almacenamiento permitido, permitiendo optimizar el espacio de bodega y asegurar la disponibilidad de piezas críticas para las reparaciones.
