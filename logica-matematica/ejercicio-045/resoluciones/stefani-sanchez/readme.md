# Ejercicio 045 - Mínimos y límites

## ¿Cómo pensé el problema?

Primero identifiqué los datos necesarios:

- Lista de servicios del taller mecánico.
- Costo de cada servicio.
- Límite mínimo y máximo permitido.

Después recorrí la lista para encontrar el servicio con menor costo y determinar cuáles servicios se encuentran dentro del rango indicado.

## Reglas aplicadas

1. Validar que existan servicios.
2. Recorrer todos los servicios.
3. Comparar costos para encontrar el mínimo.
4. Revisar cuáles costos están dentro del límite.
5. Mostrar resultados.
