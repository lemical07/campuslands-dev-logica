# Ejercicio 40 

# Explicacion 
En este ejercicio se implementa un algoritmo de simulación financiera para puestos de comida urbana. El objetivo es estructurar una estrategia que evalúe la rentabilidad de diferentes platillos, integrando costos de producción, demanda esperada y factores externos como el horario de venta, para proyectar las ganancias netas y determinar cuál de los productos en el menú aporta el mayor valor económico al puesto.
