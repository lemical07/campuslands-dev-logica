# Módulos y Divisibilidad - Festival de Cine de Terror

## Descripción
Sistema de optimización para salas de cine. Utiliza operaciones aritméticas de módulo para calcular el tiempo muerto en proyecciones y maximizar el uso de las salas.

## Reglas Aplicadas
- **Operador Módulo:** Utilizado para calcular el tiempo remanente exacto en cada sala.
- **Clasificación de eficiencia:** Lógica condicional basada en el residuo obtenido tras la división del tiempo total.

## Cómo ejecutar
El script `nombre-apellido.js` procesa un arreglo de duraciones. Ejecútalo con `node nombre-apellido.js`.

## Casos Probados
- **Perfecto:** Cuando la suma de películas es múltiplo exacto de la capacidad.
- **Desperdiciada:** Cuando sobra más de la mitad del tiempo de la sala tras las proyecciones.
