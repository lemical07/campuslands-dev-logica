# Ejercicio 42

# Explicación
En este ejercicio se implementa un algoritmo de análisis estadístico para determinar la eficiencia ofensiva de los equipos de fútbol sala. El objetivo es estructurar una estrategia que evalúe la relación entre los remates totales y los goles convertidos, calculando el porcentaje de efectividad. Además, se aplican reglas de ponderación para premiar equipos con alta efectividad frente a equipos con alto volumen de juego pero baja precisión.
