# Explicacion 
Este programa funciona como un sistema de control de calidad para la programación de saltos, garantizando que cada actividad cumpla con los estándares de seguridad establecidos por el club de paracaidismo. Evalúa automáticamente factores críticos como la altitud mínima, la velocidad del viento y la experiencia del usuario para autorizar o denegar cada salto, minimizando el riesgo operativo.
