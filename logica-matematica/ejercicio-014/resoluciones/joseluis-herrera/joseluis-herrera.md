# Ejercicio 14
# Explicación
En este ejercicio se calcula el espacio físico que ocupará un diseño de tatuaje en la piel utilizando el cálculo de áreas rectangulares. 
El objetivo es estructurar una función que multiplique las dimensiones del diseño para obtener los centímetros cuadrados totales, permitiendo al estudio clasificar el tamaño de forma automática para aplicar la tarifa de cotización correcta.
