# Ejercicio 13
# Explicación
En este ejercicio se calcula la distancia entre el punto donde cae un paracaidista y el centro exacto de la zona de aterrizaje utilizando coordenadas cartesianas. El objetivo es aplicar una fórmula matemática de distancia en una función para determinar
 si el paracaidista logró aterrizar dentro del radio de seguridad establecido o si quedó demasiado lejos.
