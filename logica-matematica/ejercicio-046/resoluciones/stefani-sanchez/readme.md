# Ejercicio 046 - Conversion de unidades

## ¿Cómo pensé el problema?

Primero identifiqué los datos necesarios:

- Nombre del auto.
- Velocidad en kilómetros por hora.
- Distancia recorrida en kilómetros.

Después definí las conversiones necesarias para transformar kilómetros a millas.

## Reglas aplicadas

1. Validar que los datos del auto sean correctos.
2. Convertir velocidad de km/h a mph.
3. Convertir distancia de kilómetros a millas.
4. Mostrar los resultados de forma organizada.

## Formula utilizada

Conversión:
millas = kilometros * 0.621371
